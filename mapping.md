Note structure:

- 0 whole note
- 1 half note
- 2 quarter note
- 3 eighth note
- 4 sixteenth note
- 5 32nd note
- 6 64th note
- 7 128th note

Any number of dots (.)

/ indicates a diddle

g indicates a grace note (we can have 1 or more grace notes)

- \> accent
- ^ housetop
- \- tenuto

Optional sticking follows (only l, r, L, or R)

t5 4{some notes} a 5:4 tuple, etc.
